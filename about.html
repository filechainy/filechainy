Introduction

Brief


Filechainy is a open-source project for transfer data using the GET method


How does the GET method work?

The method GET enables send contents to a site without fullfit a formulary or press a button. Just append to the site address the name (of the GET variable) and its contents.

How to use it?

http://test.com/index.php

Becomes

http://test.com/index.php?url=http://google.com

Or with two parameters

http://test.com/index.php?url=http://google.com&name=google

The ? is used once for separate the site address from the contents that can be sent or request.
The = symbol is used to assign to each GET variable a content as in "url=http://google.com" (where "url" is the GET variable that will receive and "http://google.com" is the content sent).
The & is utilised to separate each variable and its contents from the others

Script usage


In index.php You can use the GET parameters "hash" and "contents" or "url" for write a file (not show in search).
E.g: index.php?url=http://test.com

receive.php To send only a text or link using the GET parameter "contents" (not show in search).
E.g: receive.php?contents=123test

url.php To create a list of URLs inside a text file using the GET parameters "url" and "name" (searchable).
E.g: url.php?url=http://test.com/cat.jpg&name=cats

url_unique.php Insert a link via GET url, name, category and description to a file that will not be overwritten. (searchable).
