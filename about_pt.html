Introdução

Sobre


Nosso objetivo é criar um sistema de armazenamento descentralizado com um mecanismo de recompensas.

O modelo básico do sistema é que cada servidor deve ser capaz de verificar se os outros servidores hospedam os mesmos arquivos. Cada nó também pode comparar se os registros de arquivos são os mesmos em outros nós (ou seja, um banco de dados apenas com URLs)

O princípio do projeto é que dispositivos com a menor capacidade computacional possível possam executar um nó.

Para que isso seja possível um nodo não deve realizar todas as tarefas e nem muitas. Mas deve haver diferentes tipos de nós, por exemplo, alguns que apenas enviam arquivos, outros que hospedam ou os validam.

Um aspecto que pode tornar os nós mais rápidos e simples é, por padrão, salvar e enviar arquivos em blocos. Portanto, não há retrabalho para dividir ou montar um arquivo toda vez que ele é enviado.

Servidores bastante leves podem enviar e baixar arquivos via solicitação http GET.

Rede Descentralizada


Uma rede descentralizada deve levar em consideração:

1) A facilidade com que um nó pode participar da rede ou baixar todo o seu conteúdo

2) Salvar partes do mesmo arquivo em diferentes servidores
