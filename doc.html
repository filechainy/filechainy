Documentation

Scripts

upload.php : Upload files to the 'files' directory

filelist.php : Show the files uploaded or sent(as text)

filelist.php : Show the files uploaded (as text)

gallery.php : Show the files uploaded (as image)

urls.php : Enter manually a URL

paste.php : Simple paste tool

groups.php : Send a message to a group

zip_hash.php : Zip the contents of a folder (without the subdirectories)

zip_subdirectories.php : Zip the contents of a folder with the subdirectories

files.php : Get the path of the files in a directory by GET variable id

url.php : Send a URL using the GET variables URL and category inside a text file.

url_unique.php : Send a URL using the GET variables URL, category, name and description as a file

index.php : You can insert new items using the GET variables hash and contents. You can get the path of a file using the GET variable id (e.g: index.php?id=1) or userid, fileid and urlid respectively

convert.php : Split the files of 'file' folder in chunks of 8 Kb with a SHA1 name and save in 'hash' folder encoded in base64. A key will be created in 'key' folder with the order that the chunks of the file must be read

send.php, multiple_send.php and multiple_send_list.php : Send contents to URLs

receive.php : Receive the GET variables hash, contents and user. But shows a blank page if if the variables are not assigned

view.php : Paste the pieces of files present in a key (restore a file)

request_send.php : Send a list of links to a list of sites (URLs)

down_test.php : Download all the files presents in 'files/files.txt' using PHP

search.php : Search per files without MySQL


Java


TextSenderGUI - Send a text via GET to all urls in the 'urls.txt' file

FileCheckerV2 - Check if the files in 'hash' directory are the same of urls in 'urls.txt'

FileCheckerV3 - Check if the files in 'hash' directory are the same of urls in 'urls.txt' with latency

SendTest - Convert the file 'test.txt' to base64 and send the converted file via http GET request

SendTestV2 - Convert the files of 'files' diretory to base64 and send the converted files via http GET request

FileConverter - Convert the files of 'files' directory to base64 and split in chunks of 8kb

DownloadUrlsNew - Downloads all the urls of the text file 'urls.txt' in the 'files' directory

RemoveDuplicateUrls - Remove duplicate values of 'urls.txt'

URLGetter = Reads "files.txt" and sends each URL contained in the file to the URLs contained in "urls.txt" using the GET variable "file":

PictureViewerFull - A window that displays the images present in a text file (like a simple browser)

CheckUrls - This program read the text file 'urls.txt'and check each URL against all others for verify if they host the same files. If so, a small GET request will be performed to a log server. The files that will be checked must be in the 'hash' folder.

Filechainy.java - Default application to send and download texts via GET request to multiple hosts

UrlLatencyChecker.java - Check the latency of a list of URLs using GUI


Default directories

categories : Files uploaded or inserted via GET method

files : Other files (binary)

hash : The chunk of files with SHA1 name encoded in base64

user : Store the user that send a chunk of a file

url : URLs and servers available
